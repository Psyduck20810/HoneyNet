🔐 How Blockchain Security Works
1. Every attack captured by honeypot is serialized to JSON
2. SHA-256 cryptographic hash is computed from attack data
3. Hash + previous block hash = new block added to chain
4. Any modification to any past block invalidates ALL subsequent blocks
5. Chain integrity is verified on every read — tamper-proof audit trail

✅ Even if attacker deletes local logs — blockchain proof remains immutable
